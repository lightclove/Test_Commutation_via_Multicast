            Назначение приложения:
    Программа предназначнена для проведения испытаний в рамках изделия
БКП Технические условия КВСБ.465275.010 (ТУ)

            Требование  к персональному компьютеру (ПК):
            - две сетевых карты
            -Установленная JDK 8

            Комплектность:

    start.bat - фаил для запуска приложения;
    TestBKP.jar - исполняемый файл приложения;
    bkp.properties - конфигурационный файл;

            Yстановка программы:
    Папку  приложения разместить в файловой системе;

            Настройка конфигурационного файла:
    Используемые параметры:
    device.ip.main    - основной IP устройства(выставить в соответствии с ТУ)
    device.ip.standby - резервный IP устройства(выставить в соответствии с ТУ)
    host.ip.main      - основной IP ПК(выставить в соответствии с ТУ)
    host.ip.standby   - резервный IP ПК(выставить в соответствии с ТУ)
    port              - порт на который отправляются сообщения (выставить в соответствии с ТУ)


            Запуск приложения:
        через консоль:
            java -jar TestBKP.jar

        использую start.bat  в windows:
            дважды клиннуть по файлу start.bat

            Использование приложения:
     С помощью чек боксов выбрать используемые при тестировании сетевые интерфейсы
     Установить в полях необходимые значение:
            Длительности БИ в секундах
            Период отправки БИ в секундах
            Количество БИ 0 - бесконечая отпрвака сообщений
                               иное количество раз которое будет отправлено сообщение
            Задержка Старта -  количество секунд по истечению которых  начнется отправка сообщений с данного интерфейса
                               после нажатия кнопки старт
      Для начала тестирования нажмите кнопку "Старт"
      Для окончания теста ножмите кнопку "Стоп"

            Известные ошибки:
    При заполнении полей с параметрами Тестирование значениями отличными от натуральных чилел
    возможны сбои в работе приложений.


         Инструкции по устранению неполадок:
    1) Закрыть приложение;
    2) Проверить правильность параметров в конфигурационный файл;
    3) Запустить приложение.
